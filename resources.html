F2U Resources

Please save the images onto your device and then use an image hoster to use them! No credits needed since I looked ahead of time to make sure to post them here! Lemme know if anything changes!

You should be able to right-click and save or long-press on an image to download!

      Bookmarks

      Below are some of my favorite sites!

      • ezgif.com — A GIF converter.
      • freeformatter.com/formatters.html — Makes HTML, CSS, and JS codes neater and easier to read.
      • short.io — You can make 5 domains, 3 folders, and 1,000 "branded" links with a free account. You can delete and edit the links whenever you'd like without issues. I am unsure if you can make 1,000 "branded" links for each domain or it's just 1,000 "branded" links in total.
      • freeimage.host — Free image hoster that helps me use direct links for displaying an image on a site. Although, it bugged after too many uploads?
      • img-l.ink — Free image hoster that helps me use direct links for displaying an image on a site. This requires a Google account that creates a public shared folder for the hoster to use to make direct links.
      • netlify.com — A website hoster that can import existing GitHub projects OR you can make your own from a template or from scratch.
      • emojipedia.org — For when I'm on my laptop, it helps me copy-paste emojis directly.
      • emoji.gg — Free Discord emojis! Careful, I think I saw N*FW emojis...
      • w3schools.com — Learn how to code!
      • stackoverflow.com/questions — I look for answers on coding there!
      • gist.github.com — I sometimes host code snippets or notes there!
      • dafont.com — I download free fonts there!
      • getbootstrap.com — Formerly, when I was on Toyhouse, and I needed Bootstrap help! This includes the Charadex project made by Cheeriko that uses Bootstrap.
      • charadex.com — Cheeriko's project that I'm currently hosting 2 sites for my friend for them! Mainly used for original species creators, but you can convert it uniquely as your own site! I'm doing it now!
      • imageresizer.com — Great for resizing and cropping images.
      • pixels.crd.co — F2U tiny pixels.
      • dividers.crd.co — F2U dividers sorted by color.
      • xyz.crd.co — F2U tiny to extra-large pixels.
      • allison.crd.co — F2U code snippets for Carrd.
      • maguro.carrd.co — F2U tiny pixels, dividers, GIFs, and PNGs.
      • enchantments.carrd.co — F2U tiny pixels, backgrounds, dividers, images, and PNGs.
      • biscuit.crd.co — F2U tiny pixels, backgrounds, blinkies, GIFs, dividers, navigation decor, word art, and PNGs.
      • biscuit2.crd.co — Even more F2U tiny pixels, backgrounds, blinkies, GIFs, dividers, navigation decor, word art, and PNGs!
      • freeconvert.com — Helps with converting WEBP or unsupported file types into actual GIFs or PNGs.
      • github.com/taichikuji/discordid — Helps you advertise your Discord profile! Check the official wiki on how to do it!
      • th.circlejourney.net — For Toyhou.se users to help code out their HTML and CSS in live preview mode before publishing them publicly.
